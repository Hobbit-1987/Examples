Игра Tank_game создана для примера моего кода. Tank_game это простая диалоговая 
консольная игра в дуэль на танках в режиме вопрос-ответ. В игре есть возможность 
выбрать танк из двух типов с разными характеристиками, режим тренировки для 
повышения точности "стрельбы" и режим "бой", где проходит дуэль с вражеским танком.
